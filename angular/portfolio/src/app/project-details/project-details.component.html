Project Details

This page will provide information about the technologies used to develop each project. Feel free to check out my GitHub for the source code.


Online Shop

Technologies Used

  • Frontend: HTML, CSS & Vanilla JavaScript
  • Backend: NodeJS, Express & EJs (REST)
  • Database: MongoDB (mongoose)

Features

  • responsive
  • Authentication: signup, login, logout (password hashed with bcrypt)
  • Authorization: verification, if user is an admin (backend)


      • Admin UI

  • add products, edit products, delete products
  • order overview (all orders)


      • Non-Admin UI

  • products overview
  • view product details
  • add items to cart & remove items from cart (prices are calculated dynamically)
  • e-mail is initialized with signed up e-mail address (if logged in)
  • alternatively an e-mail needs to be provided in the shipping & payment page


Time Tracking App

Technologies Used

  • Frontend: HTML, CSS & Angular
  • Backend: NodeJS, Express & EJs (REST)
  • Database: MongoDB (mongoose)

Features

  • responsive
  • Authentication: signup, login, logout (password hashed with bcrypt)
  • track activities & keep data (if logged in)
  • remove previously tracked activities
